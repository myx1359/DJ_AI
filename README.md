# 系统概述

## 系统定位

面向计算机学院党员发展全流程管理，支持微信小程序端与PC端协同操作，覆盖入党申请人、积极分子、发展对象、预备党员、正式党员全阶段，实现组织管理、流程管控、材料归档、考核评价一体化，替代线下纸质流程，提升党建工作效率与规范性。

## 核心功能说明

**用户管理**

1.  用户管理：支持系统用户的新增、修改、删除、查询及角色、机构分配；

2.  机构管理：支持机构的新增、修改、删除、启用与停用及基础信息设置；

3.  角色管理：支持角色的新增、修改、删除及PC端、小程序端权限分配。

**党员评价**

1.  PC端数据管理：支持评价时段管理、党员评价数据导入编辑及报告生成；

2.  PC端评价结果管理：支持各类评价报告查看、导出，学院党委可撤回报告；

3.  小程序端个人信息完善：支持学生首次登录完善个人信息，可延后处理；

4.  小程序端考核评价报告：支持多角色查看报告，可筛选年份、调整排名；

5.  小程序端评价报告明细：各角色可查看对应层级的评价详情及排名。

**党员发展管理**

1.  申请人阶段：支持入党申请提交、审核及申请进度查看；

2.  入党积极分子阶段：支持积极分子确定、培育考察及材料审核流转；

3.  发展对象阶段：支持发展对象确定、培训、审核及全流程备案操作；

4.  预备党员接收阶段：支持预备党员编组及入党宣誓相关操作；

5.  预备党员转正阶段：支持转正考察、申请审核及全流程转正操作。

# 用户管理

## 用户管理

### 新增用户

进入系统后，选择【用户管理】→【用户管理】，点击"+新增用户"进入新增页面→完善用户信息→点击"确定"完成新增操作。

注：1、角色来源于【用户管理-角色管理】模块；

2.  所属组织来源于【党务管理-党群组织管理】模块；

3.  所属机构来源于【用户管理-机构管理】模块，注意先选择组织才可以选择机构；

4.  密码需同时包含大小写字母、数字和特殊字符，长度8位以上。

## 小程序端

### 个人信息完善

开放角色：学生。

学生首次登录小程序后，需要先完善个人信息，根据要求补充信息后点击提交即可。也可点击稍后处理先进入小程序，待下次登录后再完善信息。

按钮功能：

**1、前往完善个人信息：**点击前往个人信息完善页面。

**2、稍后处理：**可跳过完善个人信息步骤，先进入小程序，待下次登录后再完善信息。

**3、暂存：**暂时保存个人信息，退出该页面后个人信息将不会保存，需重新填写。

**4、提交：**所有信息按照要求填写完成后方可点击提交进入小程序。

注：个人信息关乎后续的党员发展，请务必尽早完善。

![d3aba0b7cc3351953e1cff96609868c](./images/media/image31.jpeg)

### 考核评价报告首页

开放角色：学院党委、二级支部、学生。

登录小程序后，默认进入评价页面。

考核评价报告数据来源：PC端【党员评价-评价结果】。评级报告由学院党委发布，未发布的报告、已撤回的报告均不展示在小程序中。

功能操作：

1.  **下拉选择框：**进行评价年份筛选，默认展示全部。

2.  **排名方式：**时间排名（最早、最晚）、名次排名（最前、最后），默认排名方式为时间最晚。

3.  **评价时段面板：**点击对应的评价时段即可查看评价报告明细。

    ![](./images/media/image32.png)

### 评价报告明细-学院党委

开放角色：学院党委。

**党总支概况：**展示该评价时段下的党总支名称以及学生人数。

**评价概要：**对参与该评价时段的二级支部、党员以及评价指标进行简要说明。

**支部排名情况：**默认展示该评价时段下各支部前20%的党员人数以及对比上一评价时段的排名变化情况。

![](./images/media/image33.png)

**学院内排名前20%的党员人数：**默认展示该评价时段下学院内前20%的党员情况。可根据需求切换不通指标以及排名阈值。

**党员个人排名top10：**默认展示该评价时段下各指标排名前10的党员姓名以及综合排名下该党员对比上一评价时段的排名变化情况。

![](./images/media/image34.png)

### 评价报告明细-二级支部

开放角色：学院党委、二级支部（仅可查看本支部）。

**党支部概况：**展示该评价时段下的党支部名称以及学生人数。

**评价概要：**主要展示该评价时段下党支部各评价指标在学院中的排名情况。

**支部历史排名变化：**展示所有评价时段里党支部在学院中的综合排名情况。

**学院内前20%的党员人数：**展示所有评价时段里，党支部在学院内前20%的党员人数。可根据需求切换不通指标以及排名阈值。

**支部内党员排名：**展示该评价时段下的党支部内各指标的党员排名情况以及综合排名下该党员对比上一评价时段的排名变化情况。

![](./images/media/image35.png)

### 评价报告明细-党员个人

开放角色：学院党委、二级支部（仅可查看本支部）、学生（仅可查看本人）。

**党委、二级党支部：**进入评价时段后，切换至【党员个人】页面，点击党员名字即可查看该党员的评价报告。

![](./images/media/image36.png)

**学生：**进入小程序后，点击评价时段即可查看本人在该评价时段下的评价报告。

![](./images/media/image37.png)

**基础信息：**主要展示学生姓名、学工号以及所在党支部。

**评价意见：**主要展示该评价时段下学生各指标的排名以及在学院、支部内的综合排名情况。

**评价指标排名：**主要展示该评价时段下学生各指标在学院中的排名情况。

**排名变化趋势：**主要展示该学生所有评价时段里各指标在学院、支部内的排名情况。

![](./images/media/image38.png)

#### 党支部管理员（PC端）

##### 设置基本信息

创建支部大会→导出表决票模版→填写会议基本信息→线下召开支部大会→记录会议记录和表决结果。其他组织生活会操作同理。

1.  点击"进入支部大会/支部大会"按钮进入支部大会管理，或者点击"组织生活"进入；

![](./images/media/image208.png)

2.  点击"创建组织生活"按钮，进入【设置基本信息】页面。

3.  填写会议基础信息，注意类型选择"党员大会"、是否含投票表决选择"是"、表决内容选择"预备党员转正"。其他字段可自行填写。

**补充事项**

1.  "发布范围"影响参会名单，如选择仅"支部内均可参加"，则该支部内的参与表决人员、积极分子、发展对象、预备党员、正式党员均可参加；如选择"仅党员（预备党员、党员参加）"，则只有该支部的参与表决人员、预备党员和正式党员可参加；

2.  "是否含投票表决"影响表决内容与表决人员名单，如选择"是"，则需选择表决内容、表决人员名单；如选择"否"，则无需填写表决内容、表决人员名单；

3.  点击表决人员名单的"导入人员"按钮，选择表决人员，点击"确定"按钮；

4.  点击"生成表决票"可导出表决票模版，确定基本信息填写完毕后点击"保存并发布"按钮发布会议。

##### 会议进行中

1.  会议开始后，自动生成签到码和签退码，点击对应按钮即可查看二维码。

##### 会议结束

1.  会议结束后，可根据实际参会情况，进行补录、或请假操作，未签到、签退、请假的，视为缺席。参会情况同步至参会人；

2.  点击"确认参会名单"按钮，进入组织生活记录。

##### 组织生活记录

1.  填写表决结果，点击"确定记录"按钮完成支部大会。

**补充事项**

1.  若选择预备党员的表决结果为"同意转正"，则进入下一环节；

2.  若选择预备党员的表决结果为"延迟预备期半年/延迟预备期一年"，则预备党员状态回退至上一阶段，可再次参加支部大会；

3.  若选择预备党员的表决结果为"不同意接收"，预备党员状态变为：支部大会-不通过，不可重新发展。
